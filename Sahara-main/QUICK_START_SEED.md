# 🚀 Quick Start: Fix Seed Database

## TL;DR - 3 Steps to Fix Everything

### 1️⃣ Open Admin Screen
```
App → Profile Tab → "Seed Database"
```

### 2️⃣ Clear & Re-seed
```
Tap "Clear Seeded Data" → Confirm
Tap "Seed All Data" → Wait 10 seconds
```

### 3️⃣ Restart App
```
Stop app → Run again
```

## ✅ Done!

You should now see:
- 5 services (Dog Walking, Pet Sitting, Grooming, **Training**, Vet Visit)
- 21 packages total
- 16 caregivers

## Verify It Worked

Quick test:
```
Home → Training → Should show 5 packages ✅
```

If not working, see `SEED_TROUBLESHOOTING.md`
