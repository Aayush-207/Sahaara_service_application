## How to Seed the Database

### Option 1: From Admin Seed Screen (Recommended)
1. Login to the app
2. Go to Profile
3. Tap "Seed Database" button
4. Wait for confirmation

### Option 2: Clear and Re-seed
If you want to start fresh:

1. Login to the app
2. Go to Profile
3. Tap "Clear All Data" (removes only seeded caregivers and packages)
4. Tap "Seed Database"

### Option 3: Complete Database Flush
⚠️ **WARNING: This deletes EVERYTHING including your account!**

Only use if you want to completely reset:
1. Backup any important data
2. Use the "Clear Entire Database" option
3. Re-register your account
4. Seed the database

## Troubleshooting

### Seed Button Not Working
- Check Firebase connection
- Verify Firestore rules are deployed
- Check console for errors

### Caregivers Not Showing
- Verify seed completed successfully
- Check Firestore console
- Try clearing and re-seeding

### Packages Not Loading
- Check service_packages collection in Firestore
- Verify package IDs are unique
- Re-seed if necessary
